# 🍅 番茄钟 - Pomodoro Timer

一个基于 Python tkinter 的番茄钟桌面应用，支持系统托盘、桌面通知、自定义时间设置和深色/浅色主题。

## 功能

- **番茄工作法计时** — 25 分钟工作 + 5 分钟短休，每 4 个番茄一次长休
- **圆形进度环** — 可视化剩余时间
- **桌面通知** — 时间到时弹出原生 Windows 通知
- **提示音** — 计时结束发出蜂鸣声
- **深色/浅色主题** — 内置两套配色方案
- **可配置参数** — 工作时间、短休/长休时长、长休间隔均可调节
- **窗口置顶** — 可切换是否让窗口始终保持在最前
- **番茄计数** — 记录当日完成的番茄数量

## 环境要求

- **操作系统**: Windows（提示音依赖 `winsound`）
- **Python**: 3.6+
- **依赖**: 仅使用 Python 标准库（`tkinter`、`json`、`time`、`datetime`、`os`），无需额外安装任何包

## 安装与运行

```bash
# 克隆仓库
git clone https://github.com/GCenturyAI/pomodoro.git
cd pomodoro

# 直接运行
python pomodoro.py
```

## 使用说明

| 操作 | 说明 |
|------|------|
| **▶ 开始** | 启动计时器 |
| **⏸ 暂停** | 暂停当前计时 |
| **↻ 重置** | 重置当前模式的计时 |
| **工作 / 短休 / 长休** | 手动切换模式（仅空闲时可切换） |
| **⚙ 设置** | 展开/收起设置面板 |

### 设置项

- **工作时间 (分)** — 每个番茄的时长（默认 25）
- **短休时间 (分)** — 短休息时长（默认 5）
- **长休时间 (分)** — 长休息时长（默认 15）
- **长休间隔 (个)** — 每完成多少个番茄后进入长休（默认 4）
- **窗口置顶** — 勾选后窗口始终在最前

## 配置文件

所有设置保存在 `pomodoro_config.json`，与程序同目录：

```json
{
  "work_minutes": 25,
  "short_break_minutes": 5,
  "long_break_minutes": 15,
  "pomodoros_before_long_break": 4,
  "always_on_top": true,
  "theme": "dark"
}
```

支持 `dark`（深色）和 `light`（浅色）两种主题，可直接编辑配置文件切换。
